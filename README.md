Gastronom Uploader

Устал вручную копировать рецепты из Word в форму на gastronom.ru — написал скрипты которые делают это сами.

Работает через CDP: скрипт подключается к уже открытому Chrome и заполняет форму как будто ты сам это делаешь мышкой и клавиатурой. Логин, куки, сессия — всё твоё, скрипт их не трогает.

---

Как запустить

Рецепты — двойной клик на `run.bat`

Откроется окно PowerShell и диалог выбора файла сразу в папке Загрузки. Выбираешь `.zip` или `.docx` с рецептом — дальше скрипт сам разбирает Word, находит ингредиенты в автокомплите сайта, выставляет единицы измерения и граммовку.

Подборки — двойной клик на `run_collection.bat`

То же самое но для подборок, выбираешь `.docx`.

Chrome при этом запускается автоматически если ещё не открыт. При первом запуске нужно будет залогиниться на сайте — потом логин сохраняется.

---

Что заполняется само, а что нет

Само:
- название, описание, автор
- порции, время, доп. время
- все ингредиенты с поиском по автокомплиту, единицами и граммовкой
- группы ингредиентов («Для теста:», «Для соуса:»)
- метаданные — сложность, категория, кухня, повод и т.д.
- блоки «Кстати» и «Хозяйке на заметку» выводятся в терминал — копируешь оттуда
- заполяются шаги приготовления
- загружаются картинки

Руками всё равно придётся:
- загрузить картинки к шагам (ну точнее они загружаются сами, но надо поставить источник и нажать добавить)
- нажать «Опубликовать»

Установка

Нужен Python 3.10+. Если не установлен — [python.org](https://python.org).

```bash
python -m venv venv
venv\Scripts\activate
pip install playwright python-docx pymorphy3
playwright install chromium
```

Ещё нужен файл `ingredients.json` — это база ингредиентов с сайта, по ней скрипт находит правильные названия в автокомплите. Если его нет:

```bash
python fetch_ingredients.py
```

---

Формат Word файла

Скрипт ориентируется на заголовки секций, регистр неважен:

```
Имя Автора
Название рецепта

[таблица: Порции / Время / Доп. время / Сложность / ...]

ИНГРЕДИЕНТЫ
· Ингредиент 1 — количество единица
· Ингредиент 2 — количество единица

РЕЦЕПТ ПРИГОТОВЛЕНИЯ
Шаг 1
Текст шага...

КСТАТИ
Текст блока «Кстати»
```

Группы ингредиентов обозначаются строкой с двоеточием на отдельной строке: `Для теста:`, `Для соуса:` и т.п.

Файлы

```
├── run.bat / run.ps1                       — запуск для рецептов
├── run_collection.bat / run_collection.ps1 — запуск для подборок
├── docx_to_json.py                         — парсер Word → JSON для рецептов
├── docx_to_collection_json.py              — парсер Word → JSON для подборок
├── upload_recipes.py                       — заполняет форму рецепта на сайте
├── upload_collection.py                    — заполняет форму подборки на сайте
├── fetch_ingredients.py                    — качает базу ингредиентов с сайта
└── ingredients.json                        — база ингредиентов (закоммичена)
```

`venv/`, `recipes.json`, `collections.json` и `images/` в `.gitignore` — в репозиторий не попадают.
